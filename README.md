# AI Voice Interview Agent

An AI-powered interview practice application built with Python and Streamlit.

## Features

- 🎤 Speech-to-Text using OpenAI Whisper
- 🤖 Interview evaluation using Google Gemini
- 📚 Grounded responses using a custom Q&A dataset
- 🔊 Text-to-Speech using Google TTS
- 🌐 Streamlit web interface
- 📊 Overall interview summary

## Tech Stack

- Python
- Streamlit
- OpenAI Whisper
- Google Gemini API
- gTTS

## Run

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```
